# Fine-Tuning Dataset Builder

Builds supervised fine-tuning data for educational explanations with optional Mermaid diagrams.

## Workflow

```bash
python3 dataset/generate_prompt.py
python3 dataset/merge_json.py
python3 dataset/push_to_huggingface.py
```
